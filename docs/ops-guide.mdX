## Overview
This Ops Guide documents how to run, monitor, backup, and troubleshoot the Gemini daemon and related components on macOS (developer / MVP ops).

**Default Paths**
| Component | Path |
|------------|------|
| Project Root | `~/Projects/gemini-mvp` |
| Virtual Environment | `~/Projects/gemini-mvp/venv` |
| Daemon Watch Folder | `~/Certify` |
| Ledger File | `~/Library/Application Support/gemini-ledger/ledger.jsonl` |
| Proofs Directory | `~/Library/Application Support/gemini-ledger/proofs/` |
| Signing Key | `~/.ssh/gemini/gemini_ed25519` |

---

## 1. Start / Stop / Run

### 1.1 Development Mode (Foreground)
Run the daemon directly while monitoring stdout:
```bash
cd ~/Projects/gemini-mvp
source venv/bin/activate
python cli/daemon.py
